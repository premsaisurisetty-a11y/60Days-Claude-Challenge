# Day 11 – ATS Resume Optimizer using Claude AI

## Overview

Day 11 of the ABTalks 60-Day Claude AI Challenge focused on leveraging AI for Applicant Tracking System (ATS) resume optimization.

The objective was to analyze an existing resume against a target Software Engineer Internship Job Description (JD), identify missing keywords and skill gaps, improve ATS compatibility, and generate an optimized resume that increases the chances of passing automated recruiter screening systems.

---

## Objectives

* Understand ATS resume screening mechanisms
* Analyze resume against a target JD
* Identify missing keywords and competencies
* Improve keyword alignment
* Optimize project descriptions
* Enhance professional summary
* Generate ATS-friendly resume
* Increase ATS Match Score
